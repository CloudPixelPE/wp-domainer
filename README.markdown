WP-DOMAINER
===========

A plugin for WordPress.


Usage
-----

In the post or page editor in WordPress, use the *domain* shortcode to list your domain names.

### A Simple List

[domains]
example1.com
example2.com
example3.com
[/domains]

### A List With Custom Links

[domains link="http://%dn%"]
example1.com
example2.com
example3.com
[/domains]
